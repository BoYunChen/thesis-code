		參數設定: 
		\begin{enumerate}
			\item 時間區間 $[0,25]$
			\item 原始細胞膜厚度 $h_0=0.005$
			\item 縮放倍數 $scale$
			\item 時間尺度 $\epsilon$
		\end{enumerate}
		當細胞膜增厚時，給定增厚函數為
		$$h(t)=\frac{h_0(scale+e^{-\frac{t-12.5}{\epsilon}})}{1+e^{-\frac{t-12.5}{\epsilon}}}$$
